\# NovitusApp



Aplikacja mobilna stworzona w .NET MAUI, która wyświetla podstawowe informacje o urządzeniu z systemem Android.



\## Funkcjonalności



\- Producent urządzenia

\- Model

\- Pamięć RAM

\- Pamięć wewnętrzna

\- Próba pobrania numeru seryjnego

\- ANDROID\_ID jako identyfikator



\##Technologie



\- .NET MAUI

\- C#

\- Android SDK



\##  Numer seryjny



Android 10+ blokuje dostęp do numeru seryjnego urządzenia.

Zastosowano ANDROID\_ID jako alternatywę.



\## Testowanie



\- Emulator Android

\- Fizyczne urządzenie



\## Wymagania



Aby uruchomić aplikację, wymagane jest:



\- Visual Studio 2022

\- Zainstalowany workload:

&#x20; - .NET Multi-platform App UI (MAUI)

\- Android SDK (zainstalowany razem z MAUI)

\- Emulator Android lub fizyczne urządzenie z Android 11+



\##  Uruchomienie



1\. Sklonuj repozytorium



2\. Otwórz plik `.sln` w Visual Studio



3\. Wybierz urządzenie:

\- emulator Android  

\- lub podłączony telefon



4\. Kliknij \*\*Run\*\*



\## Testowane środowisko



\- Android Emulator (API 36)

\- Fizyczne urządzenie Android
