\# NovitusApp



Aplikacja mobilna stworzona w .NET MAUI, która wyświetla podstawowe informacje o urządzeniu z systemem Android.



\## Funkcjonalności



\- Producent urządzenia

\- Model

\- Pamięć RAM

\- Pamięć wewnętrzna

\- Próba pobrania numeru seryjnego

\- ANDROID\_ID jako identyfikator



\##Technologie



\- .NET MAUI

\- C#

\- Android SDK



\##  Numer seryjny



Android 10+ blokuje dostęp do numeru seryjnego urządzenia.  

Zastosowano ANDROID\_ID jako alternatywę.
